# Testing

This package uses [SimpleTest](http://simpletest.org/) instead of PHPUnit. To run the tests:

```
$ cd tests/
$ php all_tests.php
```

# Build and install

To build, simply

```
$ pear package
```

To install from scratch

```
$ pear install package.xml
```

To upgrade

```
$ pear upgrade -f package.xml
```
